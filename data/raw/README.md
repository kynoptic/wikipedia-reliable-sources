# [`/data/raw`]

## Purpose

Raw article lists, wikitext dumps, and intermediate references fetched from Wikipedia.

## Contents

### Subfolders

* **[`wikitext/`](./wikitext/)** – Individual article wikitext files

### Files

* **[`featured_articles.json`](./featured_articles.json)** – Titles of featured articles
* **[`good_articles.json`](./good_articles.json)** – Titles of good articles
* **[`refs_raw.json`](./refs_raw.json)** – Unprocessed citation data

## Usage

Files in this directory are generated by the fetching scripts and are not meant
to be edited manually.

## Related modules

* [`../../scripts/`](../../scripts/) – Command-line utilities that populate
  these files
* [`../processed/`](../processed/) – Normalized outputs derived from the raw data
* [`../README.md`](../README.md) – Data directory overview
