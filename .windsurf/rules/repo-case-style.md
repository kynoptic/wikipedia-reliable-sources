# Enforce sentence case in headings, titles, and labels

- Do NOT use title case under any circumstance.
- Use sentence case for all headings, labels, and UI titles.
- Capitalize only the first word and any proper nouns.
- Ensure consistency across all Markdown files, UI strings, and comments.
